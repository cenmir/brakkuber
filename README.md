# Bråkkuber

Interaktiv 3D-övning i bråk. Rotera en 10×10×10-kub och räkna hur stor andel som är färglagd, uttryckt som bråk av 1 000.

## Funktioner

- **Roterbar 3D-kub** – dra för att rotera och räkna de blå ytkuberna
- **Systematisk fyllning** – kuberna fylls lager för lager från ett hörn, så att högst en sida är delvis ifylld
- **Slumpa uppgifter** – generera 5–10 nya problem med en knapptryckning
- **Svar i bråkform** – visa förkortade bråk och decimaltal
- **Avrundning** – växla mellan exakt värde, tiondel, hundradel och tusendel

## Användning

Öppna `bråk_kuber.html` i en webbläsare. Ingen installation krävs – allt är self-contained (Three.js laddas via CDN).

## Pedagogik

Övningen tränar:
- Att förstå bråk som del av en helhet (tusendelar)
- Att förkorta bråk
- Att avrunda decimaltal
- Rumslig uppfattning genom 3D-rotation
